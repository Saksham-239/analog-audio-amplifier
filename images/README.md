# Project Images and Figures

This folder contains all visual assets, schematics, and waveforms for the project. 

Please place the following files here to match the links in the main `README.md`:

1. **`final_build.jpg`**: A clear photograph of your physical breadboard prototype showing the ESP32, tone board, sensors, and amplifier circuit.
2. **`schematic.png`**: The schematic diagram of the active tone control pre-stage.
3. **`multisim.png`**: The Multisim schematic of the discrete Class-AB power amplifier stage with pre-driver.
4. **`block_diagram.png`**: The overall system block diagram showing signal flow (digital -> analog -> speaker) and control loops (ultrasonic -> volume/mute).
